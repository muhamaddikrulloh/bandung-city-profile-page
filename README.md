## Struktur Project
```
├── assets      →   Menyimpan file statis seperti gambar
│ └── image
│
├── scripts     →   Berisi file JavaScript
│ └── script.js
│
├── styles      →   Berisi file CSS
│ └── style.css
│
├── config      →   Konfigurasi website (koneksi database)
│ └── database.php
│
├── index.php   →   Halaman utama aplikasi
└── README.md   →   Dokumentasi project
```
